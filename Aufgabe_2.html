Beschreibung Aufgabe 2

Wir wollen mithilfe der Implementierung von CSS Flexbox und CSS Grid einige Bestandteile unserer Webseite verbessern. Hierbei nutzen wir CSS Flexbox in der Navigationsleiste und der Fußleiste unserer Hauptseite, wo durch sich die Elemente an die Breite des Viewports anpassen, sollte sich dieser verkleinern. In diesem Fall reihen sie sich in eine neue Zeile ein. CSS Grid wird einerseits auf der Hauptseite genutzt, andererseits aber auch auf den Unterseiten der Rituale. Auf der Hauptseite werden die Liste der Preise und das Anmeldeformular nebeneinander dargestellt, was zu Platzersparnissen führt. Auf den Unterseiten wird CSS Grid genutzt, um den Inhalt effizienter anzuordnen und so beispielsweise die Bilder oder weiteren Text neben dem Text anzuzeigen.
